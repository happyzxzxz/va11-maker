# VA-11 MAK3R

This is basically [VA-11 HALL-A](https://store.steampowered.com/app/447530/VA11_HallA_Cyberpunk_Bartender_Action/) visual novel, but in browser. You can write your own scripts using almost all characters from the original game, export your projects as json to send it to your friends or just straight up make a webm out of it with only one button.
Keep in mind that this project may have bugs.

![Screenshot](https://raw.githubusercontent.com/happyzxzxz/va11-maker/refs/heads/main/screenshots/1.png)

## Installation

1. Clone the repo
2. `cd va11-maker`
3. `npm install`
4. `npm run dev` or `npm run build` for production

## Usage

1. The Timeline: Use the left sidebar to add new frames or duplicate existing ones. Frame order is handled automatically.

2. The Inspector: On the right, select the speaker, type your dialogue, and choose which characters appear in the Left, Center, or Right slots.

3. Stage Presence: Mouth animations will automatically follow the character currently selected as the "Speaker."

4. The Jukebox: Click the Jukebox button in the timeline to assign up to 10 tracks to your session.

5. Play Mode: Press START SCRIPT to enter full-screen "Game Mode." Use Space or Click the Dialogue Box to advance.

6. Exporting: Use the Project Settings (Gear icon) to save your script to your computer or render the scene as a video file.
